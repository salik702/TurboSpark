# @craft-agent/core

Shared TypeScript types and utilities for TURBO SPARK applications.

## Usage

```ts
import type { Workspace, Session, Message, AgentEvent } from '@craft-agent/core'
import { generateMessageId, debug } from '@craft-agent/core'
```

## Contents

- Workspace and session types
- Runtime and stored message types
- Agent event types
- Shared debug and ID helpers

## Peer Dependencies

- `@modelcontextprotocol/sdk` >= 1.0.0
